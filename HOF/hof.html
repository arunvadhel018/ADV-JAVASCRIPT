Q1. Define Higher-Order Functions

A Higher-Order Function (HOF) is a function that either:
1. Takes another function as an argument
2. Returns another function as its result.

Q2. Identify HOFs

1. map() => Takes a callback function and returns a new array.[HOF]

2. forEach() => Takes a callback function to run on each element.[HOF]

3. filter() => Takes a callback to test each element and returns filtered array.[HOF]

4. console.log() => Only prints — doesn’t take or return a function. [It is not HOF]
